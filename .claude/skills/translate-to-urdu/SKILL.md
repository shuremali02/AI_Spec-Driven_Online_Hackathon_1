# Translate to Urdu Skill

## When to Use This Skill

- User says “translate this to Urdu”
- User wants Urdu explanation of English content
- User needs readable Urdu while keeping formatting and technical terms
- User wants documentation / blog / lesson translated without breaking structure

## How This Skill Works

1. **Read the English input text**
2. **Preserve structure**:  
   - Markdown headings  
   - Lists  
   - Bold / italic  
   - Code blocks  
   - Technical terms (APIs, functions, libraries)
3. **Translate naturally** into smooth, easy-to-read Urdu
4. **Return only the translated text** in clean Urdu

## Output Format

Provide:

- **Translated Urdu Text**  
  - Same formatting as original  
  - Code blocks untouched  
  - Technical terms untranslated unless context requires  

## Example

**Input:**  
"Please translate this introduction about AI agents into Urdu."

**Output:**  
"براہِ کرم AI ایجنٹس کے اس تعارف کو اردو میں ترجمہ کریں۔"
